# 欢迎来到 Vdrias World!

*Vdrias World!* 是一个相对纯净的 Minecraft Java 版生存服务器 ~

::: tip 新周目！

自 2021/7/2 以来，服务器内的众多玩家和设计师均参与到了新周目的建设当中。第四周目是新服主 Xiao_Jin 接手后从零开始的周目，在玩法上作出了一定的调整。目前，第四周目正处在公开测试阶段。

:::

- 版本： `1.17.1`
- 地址：`play.vdrias.cn / s4.vdrias.cn`
